# Simon Game

 A memory-based Simon Game built with HTML, CSS, JavaScript, and jQuery. Players must remember and repeat an increasingly complex sequence of colors and sounds.

# Features
- Random color sequence generation
- Interactive button animations
- Level progression system
- Game Over detection and restart functionality
- Responsive user interface

# Technologies Used
- HTML
- CSS
- JavaScript
- jQuery

# How to Play
1. Press any key to start the game.
2. Watch the sequence of colors.
3. Repeat the sequence by clicking the buttons.
4. The sequence gets longer with each level.
5. The game ends when you click the wrong color.

# Learning Outcomes
- DOM Manipulation
- Event Handling
- JavaScript Functions
- Arrays and Game Logic
- User Interface Design
